### 작동방식

메인 페이지에서 책 검색 버튼을 누르면 검색 페이지로 연결됨

검색 페이지에서 내용 입력 후 검색 버튼을 누르면 원하는 책 정보를 선택할 수 있는 페이지로 연결됨

검색한 책 정보 또는 비슷한 장르의 책 정보 버튼을 클릭하면 교보문고 사이트로 연결됨
